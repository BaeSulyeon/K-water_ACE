# 최적의 수처리 약품 예측 수자원공사 협업 프로젝트
최적의 응집제 주입률, 미래탁도 예측을 통해 수질의 안정성을 확보, 신속한 대응체계 구축하고 약품 사용량 절감을 통한 환경 개선을 목적으로 한다.
